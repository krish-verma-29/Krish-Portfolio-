# 🚀 Krish Verma - Portfolio Website

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript**. This project showcases my skills, education, certifications, projects, and contact information with a futuristic UI design.

## 🌐 Live Preview

> Add your portfolio live link here after deployment.

## 📌 Features

- ✨ Modern futuristic UI
- 📱 Fully responsive design
- 🌌 Animated particle background
- ⌨️ Typing text animation
- 🎯 Smooth scrolling navigation
- 🎓 Education section
- 💻 Skills section
- 📜 Certificate gallery with popup preview
- 🚀 Projects showcase
- 📧 Contact form using EmailJS
- 🎵 Background music toggle
- 📋 One-click copy for email and phone number
- 📱 Mobile-friendly navigation menu
- ✨ Scroll reveal animations

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome
- Google Fonts
- EmailJS
